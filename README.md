# ServicioSocialNLP
Este proyecto se desarrolló para el departamento de Servicio Social del Tecnológico de Monterrey para analizar la satisfacción de estudiantes en diversas Organizaciones Socio Formativas (OSFs). Los datos provienen de encuestas con respuestas abiertas y de opción múltiple, sin analizar por inconsistencias y valores faltantes.

## Logros Clave


* Construyó un modelo de red neuronal para predecir puntajes de sentimiento a partir de respuestas estructuradas de encuestas.
* Preprocesó y limpió múltiples conjuntos de datos con valores nulos, contradicciones y problemas de formato.
* Creó un tablero responsivo en Streamlit para visualizar tendencias de satisfacción en tiempo real.
* Identificó el modelo con mejor desempeño tras probar Random Forest, SVR y redes neuronales.

## Detalles Técnicos

* El modelo predice puntajes de sentimiento para preguntas abiertas clave utilizando una red neuronal con capacidad de entrenamiento en línea, permitiendo adaptabilidad a nuevos datos.
* El tablero incluye filtros, visualizaciones codificadas por color y clustering para apoyar la toma de decisiones estratégicas.

  
## Impacto
Esta herramienta permite al departamento de Servicio Social comprender mejor la experiencia estudiantil, optimizar las alianzas con las OSFs y mejorar continuamente la calidad de los programas de servicio social. Además, establece las bases para un ecosistema basado en retroalimentación, donde las tendencias de satisfacción se monitorean y actúan continuamente.
